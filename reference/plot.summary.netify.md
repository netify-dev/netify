# plot graph-level summary statistics

plot graph-level summary statistics

## Usage

``` r
# S3 method for class 'summary.netify'
plot(x, ...)
```

## Arguments

- x:

  a graph-level summary returned by
  [`summary()`](https://rdrr.io/r/base/summary.html) for a netify
  object.

- ...:

  additional arguments passed to
  [`plot_graph_stats()`](https://netify-dev.github.io/netify/reference/plot_graph_stats.md).

## Value

a ggplot object displaying graph-level summary statistics.
