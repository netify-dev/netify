# Apply style to netify plot

helper function to properly apply styles to netify plots

## Usage

``` r
plot_with_style(netlet, style_fun, ...)
```

## Arguments

- netlet:

  a netify object

- style_fun:

  a style function (e.g., style_rose)

- ...:

  additional plot parameters

## Value

a ggplot object
